Semejanzas y Diferencias entre XML y JSON

Semejanzas
- Se puede usar empleando AJAX.
- Tienen una sintáxis muy simple.
- Los valores pueden contener lista de objetos o valores (jerarquicos).
- Son independientes en el lenguaje de programación.
- Formato de texto plano.
- Legible por los humanos (autodescripción).
# Diferencias
XML - La Sintaxis es mas compleja.
- Tiene etiquetas de inicio y finalización.
- No proporciona tipos de datos y no admite matrices.
- Admite espacios de nombres.
- Admite comentarios.
- Es mejor para la configuración.
- Se pueden cambiar a otros formatos como texto sin formato.
- Está a orientado a documentos.
- Son cadenas sin escribir y deben analizarse en tiempo de ejecución.
JSON - La Sintaxis es mas ligera.
- No contiene etiquetas de inicio y finalización.
- Admite tipos de datos y matrices.
- No admite espacios de nombres.
- No admite comentarios.
- Es mejor para los servicios web.
- No se pueden cambiar a otros formatos.
- Está orientado a datos.
- Utiliza objetos escritos.
